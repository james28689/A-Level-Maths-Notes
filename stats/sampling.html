Types of Data

  • Data can be divided into two main types: qualitative and quantitative.
  • Qualitative data is non numerical, e.g: written opinions on the state of roads.
  • Quantitative data is numerical, e.g: the number of different colour cars that pass a certain point.

Discrete, Continuous, and Categoric Data

  • Quantitative data can be divided into 3 types:
    1. Continuous data is data that is numerical in nature and can take an infinite number of values.
    2. Discrete data is data that can only take certain values, and and these values are countable.
    3. Categoric data is where the data is discrete but the variables are non-numeric.
